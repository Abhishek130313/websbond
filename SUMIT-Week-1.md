## 📅 Day 1 (MON) — Lead Generation Sprint

### ☀️ Morning — Leads Collection (2 hrs)
- [ ] **Justdial Scrape:** search "web developer", "SEO services", "digital marketing" in Delhi NCR
  - Note 30 businesses with **bad or no website**
  - Add to lead sheet: Name | Phone | Website? | Called?
- [ ] **Google Maps crawl:** search "clinics in Gurgaon", "restaurants in Delhi", "salons in Noida"
  - Find 20 that have NO website listed
  - Add to lead sheet
- [ ] **Instagram search:** find 20 local businesses DM them:
  ```
  Hey! Love your page. We help businesses like yours get a website + rank on Google. Free audit?
  ```
- [ ] **Existing network:** WhatsApp 10 known contacts:
  ```  
  Bhai, Websbond launch kar diya. Website chahiye toh bata. Friends ko bhi reference de sakta hai.
  ```

### 📞 Afternoon — Call Blitz (2 hrs)
- [ ] Call 15 leads from your sheet
- [ ] Each call follow this structure:
  1. **10 sec:** Introduction + compliment
  2. **20 sec:** Problem statement (website nahi hai / Google pe nahi aati)
  3. **30 sec:** Solution (Websbond ₹7,999 website + free SEO audit)
  4. **15 sec:** Next step (demo bhejta hoon / kal call karta hoon)
- [ ] After each call → note status in tracker: Interested / Not Interested / Call Later

### 🌙 Evening — Follow-ups (1 hr)
- [ ] WhatsApp all interested leads with 1 portfolio image
- [ ] Schedule demos for interested leads (coordinate with Abhishek for demo sites)
- [ ] Send final tracker to Abhishek by 8 PM
